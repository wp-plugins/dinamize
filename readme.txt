Para agilizar seu tempo a Dinamize criou um plugin que possibilita a integração da ferramenta com seu site.

== Description ==

O Plugin da Dinamize foi criado para ajuda-lo a ter maior facilidade na instalação das funcionalidades de todas ferramentas disponíveis na Dinamize.

== Installation ==

1. Insera o conteúdo baixado na pasta `/wp-content/plugins/` na árvore de diretórios do seu site.
2. Ative o plugin na pagina de 'Plugins' no menu administrativo do seu WordPress.

Agora você tem encontrar 'Dinamize' na seu menu administrativo.

== Frequently Asked Questions ==

Esse plugin funciona tanto para Mail2Easy quanto para Mail2Easy PRO?

Não, esse plugin é compatível apenas com o Mail2Easy PRO.
